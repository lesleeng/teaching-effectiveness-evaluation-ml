# Power BI Dashboard

## Overview

This folder contains materials related to the **Power BI dashboard** developed for the Teaching Effectiveness Evaluation project. The dashboard serves as the final presentation layer, transforming analytical and machine learning outputs into **interpretable visual insights** for academic decision-making.

---

## Dashboard Purpose

The Power BI dashboard is designed to:

* Visualize **key performance indicators (KPIs)** derived from PES data
* Present trends and patterns in teaching effectiveness
* Summarize insights from numerical ratings and sentiment analysis
* Support evidence-based evaluation and instructional improvement

---

## Contents

* `dashboard_screenshots/`
  Contains screenshots of the Power BI dashboard, including KPI views, trend analyses, and summary visuals.

## Note on Power BI File

The original Power BI (`.pbix`) file is **not included** in this repository due to data privacy and institutional confidentiality considerations. Screenshots are provided to demonstrate the dashboard structure and insights.
